Projects

I have done some projects as well in python language with the names Video_GUI and Click_a_selfie. Video_GUI is a simple python program that takes the input from a link of a YouTube video and downloads and saves it to your local machine. I have used the Tkinter library in this. My other project is Click_a_selfie. It is also a simple python program that takes a picture on executing the program. I have used the OpenCV library in this.
